## 가이드 내용

### 배포 가이드

1. **사전 요구사항**
   - 시스템 요구사항
   - 필수 소프트웨어
   - 인증 설정 (API 키, 구독, 클라우드 제공자)

2. **설치 방법**
   - Claude Code CLI (npm)
   - Agent SDK (Python/TypeScript)
   - 소스에서 설치

3. **CLI 통합**
   - 커스텀 에이전트 추가
   - 에이전트 설정 옵션
   - CLI 플래그

4. **SDK 배포**
   - 기본 설정
   - 모듈로 패키징
   - PyPI를 통한 배포

5. **컨테이너 배포**
   - Dockerfile 예시
   - Docker Compose
   - Kubernetes 매니페스트

6. **CI/CD 통합**
   - GitHub Actions
   - GitLab CI
   - Jenkins 파이프라인

7. **프로덕션 고려사항**
   - 보안 모범 사례
   - 비용 관리
   - 로깅 및 모니터링
   - 헬스 체크

8. **문제 해결**
   - 일반적인 문제 및 해결책
   - 디버그 모드
   - 도움 받기
